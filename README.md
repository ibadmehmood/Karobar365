# Karobar365
This desktop applications lets you manage your business 365 days in a year very effectively with a beautiful user interface.

<b>Tools Used</b>
<ul>
  <li>C#</li>
  <li>WPF</li>
  <li>Sqlite</li>
  <li>MVVM Pattern</li>
  </ul>
